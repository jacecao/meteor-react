# meteor-react


基于node通过meteor 和 react 搭建一个简单的聊天室

整个UI来自[materialUI](http://www.material-ui.com/)框架   
meteor模拟服务器端的数据交互

另外本应用大部分代码采用ES6的书写方式  
对于需要注意的地方有注释


``` 
 npm install  //下载依赖项   
 meteor  //启动服务 运行在本地3000端口
 ```
 
 注：如果服务没有启动可能需要安装[meteor](https://www.meteor.com/)服务
